โปรเจกต์เกม "ทายว่าใคร" โดยหุ่นยนต์ AI

สิงหาคม 2024

ㆍผสานการทำงานของแขนหุ่นยนต์ KUKA สำหรับงาน BU Open House 2024 โดยเน้นการผสานเทคโนโลยีแขนกล KUKA เข้ากับความสามารถขั้นสูง เช่น การเก็บข้อมูล การฝึกอบรมด้วย Machine Learning การแปลงเสียงเป็นข้อความ และการตรวจจับวัตถุ ระบบนี้แสดงให้เห็นถึงการทำงานร่วมกันของหุ่นยนต์ AI และระบบอัตโนมัติ เพื่อสร้างระบบที่ชาญฉลาดและโต้ตอบได้
โครงการนี้มุ่งเน้นการโต้ตอบกับผู้ใช้งานเพื่อท้าทายโมเดล AI ที่ตอบสนองทั้งเสียงและภาพ แขนกล AI จะรับคำสั่งเสียงจากผู้ใช้และหยิบกล่องวัตถุที่มีลักษณะตามคำใบ้ที่ผู้ใช้อธิบายเกี่ยวกับตัวละครนั้น

การแข่งขัน Amity Hackathon 2023

มีนาคม 2023

ㆍทีมของเราได้เข้าสู่รอบชิงชนะเลิศของการแข่งขัน Amity Hackathon 2023 โดยเราได้พัฒนาแอปพลิเคชันแชทบอทที่ใช้ ChatGPT เป็นพื้นฐาน ซึ่งรวมการออกแบบส่วนหน้า (Frontend) และฟีเจอร์ API แอปนี้มีเป้าหมายในการส่งเสริมความตระหนักด้านสิ่งแวดล้อม โดยใช้ ChatGPT และ DALL-E เพื่อโต้ตอบกับผู้ใช้ในบทสนทนาเกี่ยวกับการอนุรักษ์สิ่งแวดล้อม นอกจากนี้ แอปยังมีแพลตฟอร์มชุมชนสำหรับผู้ใช้ในการโต้ตอบกัน และตลาดออนไลน์สำหรับการซื้อขายสินค้าและบริการ อีกทั้งยังมีฟีเจอร์อัปเดตสภาพอากาศแบบเรียลไทม์ เพื่อช่วยให้ผู้ใช้ได้รับข้อมูลเกี่ยวกับสภาพแวดล้อมในปัจจุบัน
โครงการนี้มุ่งเน้นการให้ความรู้และส่งเสริมให้สังคมปฏิบัติตามแนวทางที่ยั่งยืน

ผู้ช่วยสอน (Teaching Assistant) ในงาน BU Open House

ปี 2022 และ 2023

Teaching Assistant ในงาน BU Open House 2023
สอนผู้เข้าร่วมวิธีการใช้ Tableau เพื่อวิเคราะห์ข้อมูลที่ได้รับ และสร้างแดชบอร์ดเพื่อสรุปผลของงานในแต่ละรูปแบบและปัญหา
Tableau เป็นเครื่องมือที่ทรงพลังสำหรับการสร้างภาพข้อมูลและการวิเคราะห์ ช่วยเปลี่ยนข้อมูลดิบให้เป็นแดชบอร์ดที่โต้ตอบได้และให้ข้อมูลเชิงลึกที่มีประโยชน์ Tableau ถูกใช้อย่างแพร่หลายในอุตสาหกรรมเพื่อตรวจสอบและสรุปข้อมูลที่ซับซ้อนเพื่อช่วยในการตัดสินใจที่ดีขึ้น


Teaching Assistant ในงาน BU Open House 2022
สอนผู้เข้าร่วมวิธีการใช้ชุดอุปกรณ์ Makey Makey ซึ่งเป็นชุดอุปกรณ์ที่ใช้งานง่าย ช่วยให้ทุกคนสามารถเปลี่ยนวัตถุในชีวิตประจำวันให้เป็นอุปกรณ์อินพุตที่ไวต่อการสัมผัส เช่น คีย์บอร์ดหรือจอยเกม เชื่อมโยงความคิดสร้างสรรค์ เทคโนโลยี และอิเล็กทรอนิกส์พื้นฐานในรูปแบบที่สนุกสนานและโต้ตอบได้
และให้ความรู้ในบูธเทคโนโลยี Augmented Reality (AR) สามารถนำโมเดล 3 มิติมาใช้ในการสร้างสรรค์ในโลกจริงด้วย Unity ซึ่งเป็นแพลตฟอร์มการพัฒนาเกมที่ทรงพลัง ผู้เข้าร่วมจะได้เรียนรู้ว่า AR รวมเนื้อหาดิจิทัลเข้ากับโลกแห่งความจริงอย่างไร เพื่อสร้างประสบการณ์ที่น่าสนใจและโต้ตอบได้

โปรเจกต์ Machine Learning: การพยากรณ์ราคาบิตคอยน์

พฤษภาคม 2024

ㆍพัฒนาโมเดล Machine Learning (Linear Regression, Random Forest, Gradient Boosting) เพื่อพยากรณ์ราคาปิดของบิตคอยน์ โดยดำเนินการเตรียมข้อมูลล่วงหน้า (Preprocessing), ประเมินประสิทธิภาพ (MSE, R') และแสดงผลด้วย Python, Pandas, Scikit-learn และ Matplotlib
ความแม่นยำของโมเดล: บรรลุความแม่นยำสูงด้วย LSTM สำหรับการพยากรณ์ระยะสั้น ในขณะที่ Random Forest มีประสิทธิภาพสูงสำหรับการพยากรณ์ระยะกลางด้วยชุดข้อมูลที่มีคุณลักษณะหลากหลาย

การแสดงผล: แสดงผลการพยากรณ์อย่างชัดเจนด้วยแนวโน้มราคาบิตคอยน์, ตัวชี้วัดทางเทคนิค และผลลัพธ์การพยากรณ์

การประยุกต์ใช้ทางธุรกิจ: ช่วยให้ผู้ค้าสามารถระบุแนวโน้มและตัดสินใจได้อย่างมีข้อมูล อีกทั้งยังสามารถปรับใช้สำหรับการพยากรณ์แบบเรียลไทม์ด้วยการปรับเปลี่ยนเพียงเล็กน้อย

โปรเจกต์บัญชีทางการ LINE และแชทบอท

พฤศจิกายน 2023

ㆍโปรเจกต์นี้ผสานการทำงานของบัญชีทางการ LINE (LINE Official Account) เข้ากับเว็บแอปพลิเคชันสำหรับการจองการท่องเที่ยว และเชื่อมต่อกับ Chat-GPT API เพื่อให้บริการที่เกี่ยวข้องกับการท่องเที่ยวแบบโต้ตอบ ผู้ใช้สามารถเรียกดูและจองแพ็คเกจท่องเที่ยว รวมถึงสอบถามข้อมูลเกี่ยวกับการท่องเที่ยวผ่านระบบแชทบอทที่ใช้งานง่าย
การผสานระบบนี้สร้างแพลตฟอร์มที่ทันสมัยและเป็นมิตรกับผู้ใช้ โดยผสานการเข้าถึงการสื่อสารของ LINE เข้ากับการโต้ตอบที่ขับเคลื่อนด้วย AI และระบบจองการท่องเที่ยวที่มีประสิทธิภาพ เหมาะสำหรับนักเดินทางยุคใหม่อย่างยิ่ง!

ความสามารถและทักษะ

คลิกที่ไอคอนเพื่อดูรายละเอียด

เชี่ยวชาญการใช้ไลบรารี Python เช่น Matplotlib, Pandas, OpenCV ฯลฯ พร้อมความเข้าใจในโครงสร้างข้อมูลและอัลกอริทึม

มีประสบการณ์ในการสร้างแดชบอร์ด, การสร้างภาพข้อมูลแบบโต้ตอบ และการใช้ Tableau ในการเล่าเรื่องและวิเคราะห์ข้อมูล

มีประสบการณ์ในการใช้ LookML สร้างรายงานที่ปรับแต่งเอง และการสร้างภาพข้อมูลที่ขยายขนาดได้

มีประสบการณ์ในการใช้เครื่องมือด้านความปลอดภัยของ Palo Alto เช่น การจัดการเครือข่ายไฟร์วอลล์ และหลักการความปลอดภัยของคลาวด์เบื้องต้น

มีประสบการณ์กับบริการ Google Cloud เช่น Compute Engine, Cloud Storage, BigQuery และการสร้างโซลูชันคลาวด์ที่ขยายได้

เชี่ยวชาญการเขียนคำสั่ง SQL ซับซ้อนเพื่อการดึงข้อมูล การจัดการ และการปรับปรุงฐานข้อมูล พร้อมความเข้าใจเรื่อง joins, subqueries และการเพิ่มประสิทธิภาพ

มีประสบการณ์ในการสร้างและจัดการคอนเทนเนอร์ การสร้างภาพ Docker และการปรับใช้แอปพลิเคชันที่สามารถขยายได้ด้วย Docker

เชี่ยวชาญในการออกแบบ จัดการ และปรับปรุงฐานข้อมูล MySQL พร้อมความเข้าใจในแนวคิดฐานข้อมูลเชิงสัมพันธ์

มีประสบการณ์ในการวิเคราะห์ข้อมูล การสร้างโมเดลข้อมูล และการวิเคราะห์เชิงพยากรณ์ด้วย IBM SPSS รวมถึงการประมวลผลข้อมูลจากการสำรวจ

มีประสบการณ์ในการออกแบบส่วนติดต่อผู้ใช้ การสร้างต้นแบบ และกระบวนการทำงานร่วมกันด้าน UI/UX ด้วย Figma

มีพื้นฐานที่แข็งแกร่งใน HTML5 การสร้างหน้าเว็บที่มีโครงสร้างและความหมายสำหรับการพัฒนาเว็บสมัยใหม่

มีความชำนาญใน CSS3 สำหรับการตกแต่งหน้าเว็บ การออกแบบที่ตอบสนอง (responsive design) การสร้างแอนิเมชัน และความเข้ากันได้กับเว็บเบราว์เซอร์ต่าง ๆ

มีทักษะในการใช้ไลบรารี Python เช่น Matplotlib, Pandas, OpenCV ฯลฯ พร้อมทั้งมีความเข้าใจในโครงสร้างข้อมูลและอัลกอริธึมอย่างลึกซึ้ง

มีทักษะในการสร้างแอปพลิเคชันแบบหน้าเดียว (Single-page applications) โดยใช้ React

การพัฒนาแอปพลิเคชันเว็บที่สามารถขยายขนาดได้ด้วย Ruby on Rails โดยมุ่งเน้นที่สถาปัตยกรรม MVC และ RESTful APIs

มีประสบการณ์ในการสร้างโมเดล 3D และการออกแบบด้วย FreeCAD พร้อมความรู้เกี่ยวกับกระบวนการออกแบบพารามิเตอร์

มีทักษะในการใช้ Roboflow สำหรับกระบวนการวิจัยคอมพิวเตอร์วิชัน การเตรียมข้อมูล และการสร้างชุดข้อมูลที่พร้อมสำหรับการเรียนรู้ของเครื่อง

มีความชำนาญในการตัดต่อภาพ การออกแบบกราฟิก และการตัดต่อวิดีโอโดยใช้ Adobe Photoshop และ Premiere Pro
